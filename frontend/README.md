# Girdhar Trading Company - Landing Page

A modern, professional landing page for Girdhar Trading Company - a commission agent and grain retailer.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (custom grain/earth color palette)
- **Framer Motion** (smooth scroll animations)
- **Lucide React** (professional icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

Or export as static site:
```bash
npm run build
```

The static files will be in the `out/` directory.

## Features

- **Hero Section** - Full-screen hero with grain market imagery
- **About Section** - Company information and mission
- **Services Section** - Grain retail and commission agent services
- **Contact Section** - Clickable phone, email, WhatsApp, and address
- **Footer** - Social media links and copyright
- **Responsive Design** - Mobile-friendly layout
- **Smooth Animations** - Fade-in effects on scroll
